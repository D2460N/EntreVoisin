# EntreVoisins

  Est une application qui permet à des personnes d’un même quartier de se rendre des petits services.
  
# Fonctions présentes

- Lister mes voisins
- Ajouter un voisins
- Supprimer un voisins

# Fonctions ajoutées

- affichage détaillé du voisin
- création d'une liste de  voisins favoris
